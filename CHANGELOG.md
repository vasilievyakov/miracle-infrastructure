# Changelog

## 1.0.0 (2026-02-21)

Initial release. 15 skills across 7 packs, battle-tested across 1169+ sessions.

### Memory Pack
- `session-save` - Save session context to project memory
- `search-memory` - Query observations with progressive disclosure
- `memory-health` - Validate memory system integrity
- `memory-init` - First-run setup wizard
- `project-status` - Quick project status from dossier + git
- 3 rules: `session-start`, `session-end`, `auto-observe`

### Thinking Pack
- `directors` - 5-director virtual advisory board
- `frameworks` - 50 frameworks organized by project stage
- `orchestrate` - 2-4 parallel specialized agents

### Research Pack
- `researching-web` - Web research with confidence scores
- `triangulate` - 3+ source fact verification
- `learned-lessons` - Knowledge base from solved problems

### Business Pack
- `transcript-to-proposal` - Call transcript to commercial proposal + prototype

### Content Pack
- `action-items` - Extract tasks from transcripts and documents

### Productivity Pack
- `aqal-review` - Weekly integral review across 4 quadrants

### Meta Pack
- `skill-checkup` - Skills library health audit
